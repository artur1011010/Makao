# Makao

Browser card game, based on REST

## Built With
* Java 11
* [Maven](https://maven.apache.org/) - Dependency Management
* [Spring Framework 5](https://spring.io/) - Spring boot, security, data, mvc
* [Bootstrap 4](https://getbootstrap.com/)
* JavaScript
